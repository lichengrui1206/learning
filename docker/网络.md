### 容器网络互联

#### 1.创建一个网络

```nginx
docker network create 姓名
```

#### 2.将指定容器加入

```
docker network connect 网络名称 容器名称
```

#### 3.也可以在创建容器时直接加入网络

```nginx
docker run -d --name dd -p 8080:8080 --network 网络名称 容器名称
```
